Prework Study Guide

HTML

  • The head element contains info about the webpage
  • The body element represents the visible content shown to the user.
    • section elements are semantic elements that define a section in a document that contains thematic content.
  • HTML uses the id attribute as a way to identify elements and select them.

CSS

  • Three ways to style a webpage using CSS: inline, internal, and external CSS styling sheet
  • CSS rule: the selector defines the element or attribute. Declaration has the CSS property and value.
  • Box Model: contains margin, border, padding, and content.
  • Wild card symbol *: declares ALL elements on the page will have the style.
  • Add the link element in the header sectiopn, below the last meta element and before the title element.
  • Class attributes: .card designates this as a class selector. You can add the class in the sections of the HTMK code: section class="card"

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin main: pull down the latest main branch
